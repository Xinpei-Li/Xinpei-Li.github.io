# 🔮 萬能預言家之書 (The Omniscient Oracle Book) > 輸入你的任何問題，讓命運之神給你一個既荒謬又令人深思的答案！ ## 🌟 專案簡介 這是一個基於 Web 前端技術開發的互動式問答小遊戲。本專案旨在結合神祕的占卜主題與輕鬆幽默的網路文化，提供一個簡單有趣的線上互動體驗。 ## ✨ 主要特色 * **神秘與趣味結合：** 採用古老的書籍風格設計，但答案庫充滿了無厘頭的幽默回答。 * **互動儀式感：** 點擊按鈕後，模擬 1.5 秒的「命運運算中」加載動畫，增加揭曉答案時的期待感。 * **純前端實現：** 僅使用 HTML, CSS, JavaScript 實現所有功能，無需後端資料庫。 * **響應式設計：** 基礎樣式已適配手機和桌面瀏覽器。 ## 🛠️ 技術棧 | 技術 | 目的 | | :--- | :--- | | **HTML5** | 建立網頁結構與輸入介面。 | | **CSS3** | 頁面美化、排版，包含動畫 (`@keyframes`) 與主題風格設計。 | | **JavaScript** | 隨機數生成、答案選擇、DOM 操作、延時控制 (`setTimeout`)。 | ## 🚀 如何開始 (部署與運行) ### 1. 本地運行 1. 克隆 (Clone) 本儲存庫到您的電腦： ```bash git clone [您的儲存庫 URL] ``` 2. 進入專案目錄： ```bash cd oracle-book ``` 3. 直接用任何瀏覽器（如 Chrome, Firefox）開啟 `index.html` 檔案即可。 ### 2. 線上部署 (GitHub Pages) 您可以將本專案部署到免費的 GitHub Pages 服務： 1. 將 `index.html` 和 `script.js` 上傳到您的 GitHub 儲存庫。 2. 在儲存庫 **Settings** 中啟用 **Pages**，選擇 `main` 分支作為來源。 3. 稍等片刻，您的專案即可透過 `[您的用戶名].github.io/[儲存庫名稱]` 訪問。 ## 💡 貢獻與優化 歡迎對這個專案進行貢獻或提出優化建議！您可以： 1. **擴充答案庫：** 在 `script.js` 的 `answers` 陣列中加入更多爆笑的預言。 2. **CSS 美化：** 增加更豐富的紋理、動畫或響應式佈局。 3. **UI/UX 改進：** 增加輸入文字的動畫效果，或更換 loading 動畫。 ---
